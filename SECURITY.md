# Security Policy

If you believe you've identified a security vulnerability in Tayyar / Mastodon (a bug that allows something to happen that shouldn't be possible), please report it privately.

- **For Tayyar-specific issues**: Contact 7X Corporation via GitHub
- **For upstream Mastodon issues**: Open a [GitHub security issue on the Mastodon project](https://github.com/mastodon/mastodon/security/advisories/new) or reach them at <security@joinmastodon.org>

You should _not_ report such issues on public GitHub issues or in other public spaces.

## Scope

A vulnerability in the code distributed through our main source code repository on GitHub. Vulnerabilities that are specific to a given installation (e.g. misconfiguration) should be reported to the owner of that installation.

## Supported Versions

| Version | Supported        |
| ------- | ---------------- |
| 4.6.0   | Yes              |
| 4.5.x   | Yes              |
